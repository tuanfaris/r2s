# NanoPi R2S ImmortalWrt 25.12 Premium Theme V3 Fixed

This build keeps the official LuCI Bootstrap theme engine and appends premium CSS after the original layout CSS.
It avoids the broken plain-list layout caused by replacing `cascade.css` completely.

Target:
- NanoPi R2S
- ImmortalWrt 25.12.0-rc2
- rockchip/armv8
- friendlyarm_nanopi-r2s

Build using GitHub Actions, then flash the `.img.gz` artifact to SD card.
